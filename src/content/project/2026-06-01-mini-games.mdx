[Mini Games](https://games.mazipan.space/) is a curated collection of classic browser games — all free, all playable instantly, no account or download required. Each game is a self-contained page built with vanilla JavaScript and styled with Tailwind CSS.

Classic games like Snake, Tetris, and 2048 are timeless, but most online versions come wrapped in ads, paywalls, or bloated apps. Building a clean alternative for all of them at once would have taken me much longer on my own. With AI assistance, I was able to implement the game logic for 19 different games — something that was close to what I wanted but previously hard to pull off solo. This is the stripped-down version: just the games, in your browser, as fast as the page loads.

## ✨ Features

- **Arcade games** — Flappy Bird, Pong, Snake, Whack-a-Mole, Breakout, Tetris, Space Invaders
- **Puzzle games** — 2048, Sliding Puzzle, Minesweeper, Sudoku
- **Board games** — Tic-Tac-Toe, Memory Match, Blackjack, Connect Four, Mancala
- **Word games** — Hangman
- **19 games total** across 4 categories
- **No login required** — every game is playable immediately
- **No ads, no tracking** — clean experience, nothing phoning home

## 🗂️ Tech Stacks

- Vanilla JavaScript
- Tailwind CSS v4
- Eleventy 3.x
- Nunjucks
- Bun
