[Ajari Koding](https://dub.sh/ajari-koding) is a curated directory of Indonesian coding learning resources, maintained by the [PHPID community](https://www.facebook.com/groups/phpid/).

One of the most common questions in Indonesian developer communities is: "mau belajar coding dari mana?" (where do I even start learning to code?). There's no shortage of resources — the problem is finding the trustworthy ones. YouTube channels, blogs, and course sites in Bahasa Indonesia vary wildly in quality, and beginners have no reliable signal for what's worth their time.

Ajari Koding started as a community effort to solve exactly that: a curated list where every entry has been vetted by real PHPID members and given an honest rating. No spam, no self-promotion without scrutiny. Just the best Indonesian coding resources, organized so anyone can find a starting point without getting lost.

## ✨ Features

- **70+ vetted resources** — hand-picked and community-rated entries, not raw link dumps
- **Multiple content types** — videos, written tutorials, online courses, blogs, portals, and eBooks
- **Community ratings** — each resource is rated 1–5 stars by PHPID members so learners can prioritize quality
- **Standardized entries** — every resource includes creator info, URL, description, topics covered, and business model (free/paid)
- **Broad topic coverage** — web dev (HTML, CSS, JS, PHP), backend frameworks (Laravel, Django), mobile (Flutter, Android), DevOps, data science, and more
